# BBjReact

BBjReact is a widget that allows to create web UI

&nbsp;

### Load a JSON file

Loading can be done with both ReactWindow and ReactChildWindow classes

**e.g.**

```java
new ReactWindow().loadFromJson("myFile.json")
```

### Demo

-   Demos are available in folder **/demos**
-   In case of unfound files, try to add project folder in **config.bbx** configuration file

&nbsp;

### Development mode for React components

**You can edit/create web components following this part**

1. Setup

    - Make sur **Node.js** is installed on your computer
    - Go to _client_ folder and run `npm install` to install all dependancies

2. Usage
    - By default React component files are located in **/client/src** folder
    - You can create your own component editing **/client/src/components.jsx** (_or you can create a new one_), then make sure to import this custom component in **/src/client/index.js** and to add it into **modules** array.
      &nbsp;
        ```jsx
        import * as controls from './components.jsx'
        import { MyCustomComponent } from './customComponent.jsx'
        const modules = [controls, MyCustomComponent]
        ```
    - Run `npm run build` to create a javascript build
    - By default, output file will be created in **/client/dist/main.js** and will be automatically loaded by **BBj**
    - However, you can edit **.js** built file injection in **/BBjReact/ReactContainer.bbj**
      &nbsp;
        ```java
        void onInit(BBjEvent ev!)
        ```

&nbsp;

### **Tips**

-   You can increase performance using _relative_ paths (**Json**, **Js**, **BBj**, ...)
-   Webpack hint : performance can be increased avoiding too much modules

### **Json structure example _with bootstrap CSS_**

Go to **/demo/demo_json.bbj** to see the result

```json
{
    "type": "div",
    "class": "container mt-5",
    "children": [
        {
            "type": "ReactButton",
            "class": "btn btn-primary mb-5",
            "text": "ReactButton with Bootstrap CSS"
        },
        {
            "type": "ReactInputN",
            "class": "form-control mb-5",
            "placeholder": "Type something..."
        },
        {
            "type": "ReactListButton",
            "title": "title",
            "class": "form-control mb-5",
            "options": ["a", "b", "c"]
        },
        {
            "type": "ReactTabCtrl",
            "children": [
                {
                    "type": "ReactChildWindow",
                    "name": "Page 1",
                    "children": {
                        "type": "ReactStaticText",
                        "text": "Text in page 1"
                    }
                },
                {
                    "type": "ReactChildWindow",
                    "name": "Page 2",
                    "style": {
                        "textTransform": "uppercase",
                        "color": "blue"
                    },
                    "children": {
                        "type": "ReactStaticText",
                        "text": "Uppercased blue text in page 2"
                    }
                }
            ]
        }
    ]
}
```
